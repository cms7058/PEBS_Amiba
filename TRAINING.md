# Amoeba Copilot 一日培训大纲

> 阿米巴动态智能体系统 · 上海零参科技
> 时长：1 天（上午 3 小时 + 下午 3.5 小时）
> 对象：阿米巴顾问、企业经营管理者、财务/IT 对接人
> 形式：讲解 40% + 实操 50% + 答疑 10%

---

## 培训前准备（提前 1 天发给学员）

| 准备项 | 说明 |
|---|---|
| 账号 | 每位学员一个登录账号（邮箱+邀请码，或讲师预建的 consultant 账号） |
| 设备 | 每人一台笔记本（Chrome / Edge 浏览器）+ 手机（扫码体验移动端） |
| 环境 | 培训用服务器地址 + 至少 1 个已配好的 LLM API Key |
| 预习 | 《阿米巴动态智能体系统设计方案》第一、三章（系统定位 + 六维诊断模型） |
| 材料 | 操作手册 MANUAL.md 打印版或电子版、一页宣传图 poster.pdf |

# 上午场（9:00 – 12:00）· 从理念到诊断

## 模块 1 · 开场与方法论（9:00 – 9:40，40 分钟，讲解）

**目标：让学员理解"为什么需要动态智能体"，而不是先学按钮在哪。**

- 阿米巴落地的三大现实挑战：制度层 / 信息化层 / 设备物理层
- 传统咨询模式的痛点：贵（50-300 万）、慢（1-2 年）、模板化（企业套模板）
- 动态陪跑 vs 静态交付：四大能力（诊断 / 设计 / 陪跑 / 进化）
- 产品全景：四层引擎流水线（诊断 → 规划 → 设计 → 部署）
- 案例引入：四家演示企业的不同阶段（待诊断 / 诊断完成 / 设计中 / 部署中）

> 教具：一页宣传图（poster.pdf）+ 总览页大屏演示

## 模块 2 · 登录与系统导览（9:40 – 10:10，30 分钟，实操）

**实操 ①：人人登录 + 走一圈**

1. 打开登录页，体验两种登录方式
   - 邮箱 + 邀请码（正式通道）
   - 管理员密码（运维通道，仅讲师演示）
2. 总览页 5 分钟导览：统计卡 / 四引擎入口 / 最近企业 / 系统预警
3. 点击"最近服务企业"中的 **宁波恒展** → 进入企业 demo 详情页
   - 看 KPI 磁贴、六维雷达、12 个月经营趋势
4. 回到总览 → 点击一条系统预警（如"苏州智微里程碑延误"）
   - 看预警钻取页：触发趋势 / 影响范围 / 时间线 / 行动建议
5. 手机扫总览页二维码 → 移动端登录 → 浏览 `/m` 页面

✅ 检查点：每位学员都能登录、能在 PC 和手机之间切换查看。

## 模块 3 · 六维诊断引擎（10:20 – 11:30，70 分钟，重点实操）

**讲解（15 分钟）：**
- 六维模型：组织 / 财务 / 信息化 / 设备 / 流程 / 文化
- 每维 L1 表层 → L2 中层 → L3 深层的问询逻辑
- 卡片式问答的设计意图：让老板"答题"而不是"写作文"
- AI 如何动态出题：题库 + 企业上下文 + 长期记忆

**实操 ②：完整跑一次真诊断（45 分钟）**

> 学员两人一组：一人扮演"企业老板"（按发的企业背景卡回答），一人操作系统。

1. 诊断引擎 → 「新建企业」（填名称 / 行业 / 规模）
2. 「新开诊断」→ 开始卡片问答
   - 体验 5 种题型：单选 / 多选 / 数字 / 短文本 / 长文本
   - 观察右侧六维进度条实时变化、维度完成的绿色 ✓
3. 中途故意关闭浏览器再打开 → 验证对话自动恢复（持久化）
4. 答到 AI 给出完成卡片 → 观察自动生成画像
5. 右上「导出 MD」→ 打开导出的诊断报告，看结构

**讨论（10 分钟）：**
- 你扮演老板时，哪些问题"问到痛处"了？哪些问题想跳过？
- 真实客户访谈中如何配合智能体（人机协作的边界）

✅ 检查点：每组完成一份诊断、产出一份 MD 报告。

## 模块 4 · 企业画像深读（11:30 – 12:00，30 分钟，实操）

**实操 ③：把刚生成的画像讲给"老板"听**

1. 企业画像页 → 选择自己刚诊断的企业
2. 逐区域解读：
   - 综合就绪度 / 推荐档次 / 预计周期 → 这三个数怎么向客户解释
   - **阿米巴落地全景思维导图**：7 大分支（现状/去除/改造/新建/规则/步骤/风险）
     - 点击节点 → 看详情卡 + 嵌入图表
     - 颜色编码：灰=沿用 / 红=去除 / 琥珀=改造 / 蓝=新建 / 紫=规则
   - 六维诊断神经图：hover 看关联、点击看 L1/L2/L3 子问题
   - 问题关联图：优势 → 风险 → 维度 → 决策点 的因果链
   - 同行业基准对比：百分位的话术（"超过 67% 的同行"怎么讲）
3. 每组选 1 人，用 3 分钟把画像讲成"给老板的汇报"

✅ 检查点：学员能用画像页完成一次 3 分钟客户汇报。

# 下午场（13:30 – 17:00）· 从规划到落地陪跑

## 模块 5 · 规划引擎：把差距变成规章（13:30 – 14:30，60 分钟）

**讲解（15 分钟）：**
- 规划引擎的定位：画像找出"差什么"，规划回答"怎么补"
- 六个 Tab 的逻辑顺序：数据采集 → 制度 → 流程 → 核算规则 → 风控 → 节奏

**实操 ④：为自己的企业定制规章（35 分钟）**

1. 数据采集计划 Tab：11 类数据源逐条看（IoT/MES/ERP/手工/钉钉/新建）
   - 点击一行 → 右下 AI 助手同步该上下文
2. 核算规则 Tab：精读 2 条公式
   - 内部转让定价：`转让价 = 标准工时成本 × (1 + 内部利润率%)`
   - 辅料定额消耗法 + 超额自承担规则
3. 业务流程 Tab：展开「月度阿米巴核算与对账」SOP，理解 6 步责任人
4. 防风险措施 Tab：风险矩阵的读法（右上=优先治理）
5. **AI 助手实战**：点击「内部转让定价公式」→ 打开 AI 助手 →
   - 点「生成 CSV 模板」→ 下载工作底稿
   - 追问"我们有 5 个阿米巴，帮我扩展这张表" → 观察 AI 用上下文细化
   - 点「找相关资料」→ 看学习方向推荐

**讨论（10 分钟）**：哪条规则在你的企业最难落地？为什么？

✅ 检查点：每人下载至少 1 份 AI 生成的 CSV 工作底稿。

## 模块 6 · 设计引擎与部署引擎（14:30 – 15:40，70 分钟）

**实操 ⑤：读懂切割与定价（30 分钟）**

1. 设计引擎 → 切换 4 家企业，对比切割差异
   - 汽车零部件：按工序切（冲压/焊接/涂装/总装）
   - 项目制非标：按项目切 + 资源池
2. 阿米巴切割神经树：点 HQ / 分类 / 单元三层节点
3. KPI 表：目标 vs 实际 双值徽章怎么读
4. 内部转让定价矩阵：行=卖方 列=买方，每格的公式来源

**实操 ⑥：部署引擎的运行台账（30 分钟）—— 全天最重要的页面**

1. 部署引擎 → 选「宁波恒展」→ 展开「冲压阿米巴」一行
2. 完整走一遍展开面板的 11 个区块：
   - 成本结构 donut + 成本拆解
   - 员工工时表（折合时薪怎么算）
   - 设备 OEE 三色 / 能耗明细 / 辅料定额对比
   - **产品产出与价值**（附加值率的含义）
   - **浪费成本明细**（7 类浪费 + 根因 + "可消除部分"）
   - 核算规则 / 治理规则 / 数据架构图（点架构图节点看字段与对接方式）
   - 优点 / 缺点 / 改进方向
3. 对照「涂装阿米巴」（问题单元）再看一遍：辅料超耗 14% 在哪些区块留下痕迹
4. 阿米巴关系神经图：超预期(绿)/正常(蓝)/需关注(琥珀) 的全局视角

**讨论（10 分钟）**：如果你是涂装阿米巴长，看完面板第一件事做什么？

✅ 检查点：学员能从展开面板中找出"浪费最大的 1 项 + 它的根因 + 改进项"。

## 模块 7 · 管理员功能（15:50 – 16:20，30 分钟，admin 角色实操）

> 仅企业内的系统管理员 / 主带顾问需要，其他学员观摩。

**实操 ⑦：三件管理事**

1. 用户管理：新建一个 viewer 账号 → 用它登录看权限差异
2. 题库与建议：
   - 题库 Tab：看 18 道内置题的 6 维 × 3 层结构，新增 1 道自定义题
   - 待审建议 Tab：演示 AI 提出的新题如何审批入库（**AI 永不自动入库**）
3. 模型与设置：
   - 三个模型（DeepSeek / MiniMax / Kimi）的协议差异（OpenAI vs Anthropic）
   - 换默认模型 → 测试连接 → 回诊断引擎验证生效

✅ 检查点：admin 学员完成一次"建号 + 加题 + 切模型"全流程。

## 模块 8 · 综合演练 + 结业（16:20 – 17:00，40 分钟）

**结业演练（25 分钟）：모拟客户全流程**

每组抽一张"企业情景卡"（讲师准备 3-4 种行业情景），限时完成：

1. 新建企业 + 快速诊断（答 10-15 题即可手动「标记完成」）
2. 在画像页选出 **3 个最关键发现**（风险/决策点）
3. 用 AI 助手为其中 1 个发现生成落地工作底稿（CSV）
4. 2 分钟小组汇报：就绪度多少 → 最大风险是什么 → 第一步做什么

**评分维度**：流程完整性 / 画像解读准确性 / AI 助手使用熟练度 / 汇报说服力

**结业答疑 + 行动计划（15 分钟）**

- 高频问题集中答疑（参照 MANUAL.md FAQ）
- 每位学员写下"回去后 7 天内要做的 3 件事"
- 发放资料包：MANUAL.md + 设计方案 + 今日产出的报告与 CSV
